# Installation
```
    1. git clone https://github.com/yousefmrashad/checkdata
```    
```    
    2. code checkdata
```       
```    
    3. <make your changes to <checkdata.ipynb> file>
```    
```    
    4. git pull
```    
```    
    5. git add . 
```    
```    
    6. git commit -m "<your commit message>"
```       
```    
    7. git push -u origin main 
```
